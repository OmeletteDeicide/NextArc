# NextArc 🎬

Application mobile **Flutter** de suivi et de recommandation d'animes.
Connecte-toi avec ton compte **AniList** pour retrouver ce que tu as déjà vu,
tes notes et tes favoris — et découvre quoi regarder ensuite.

## ✨ Fonctionnalités (prévues)

- 🔥 Découverte : animes tendance et de la saison en cours
- 🔎 Recherche par titre
- 📄 Fiche détaillée (synopsis, score, genres, épisodes)
- 🔐 Connexion AniList (OAuth) : liste perso, progression, notes, favoris
- 🤝 Recommandations basées sur ce que tu aimes
- 🌙 Thème sombre

## 🛠️ Stack technique

- **Flutter** (Dart)
- **Riverpod** — state management
- **GraphQL** — API [AniList](https://anilist.co)
- **Hive** — stockage local
- **Firebase** — données utilisateur (optionnel)
